# Tutti OS Repository Template

Use this template for new Tutti OS open-source repositories.

It includes:

- local `AGENTS.md` instructions for AI coding agents;
- local `CONTRIBUTING.md` pointing to organization-wide contribution rules;
- a lightweight PR title check workflow.

After creating a repository from this template, update `README.md` and
`AGENTS.md` with project-specific setup, test, and release instructions.
